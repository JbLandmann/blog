# JB Landmann — Blog

Blog personnel sur l'AI, l'automatisation et les outils de développement.

🔗 **[jblandmann.github.io/blog](https://jblandmann.github.io/blog)**

## Stack

- [Astro](https://astro.build) 4 — Static site generation
- MDX — Articles enrichis
- GitHub Pages — Hébergement

## Développement

```bash
npm install
npm run dev      # Serveur local http://localhost:4321
npm run build    # Build de production
npm run preview  # Preview du build
```

## Déploiement

Push sur `main` → GitHub Actions build + deploy automatique.
